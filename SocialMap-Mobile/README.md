## Inicie o projeto Mobile (Em desenvolvimento)
```shell
$ cd SocialMap-Mobile
$ npm install
```

### Mudar o valor no arquivo constante.ts para o host da sua internet  
```js
const CONSTANTS = {
    API_HOST: 'http://SUA_REDE:4000/v1',
    SOCKET_HOST: 'http://SUA_REDE:4000/v1'
}
```
### Rodar o projeto
```shell
$ npm start
```
